As usual, I'll start by exploring the main function.

Putting it all together

The user input is compared against many indivual characters.

If we concatenate all these characters, we obtain the flag HTB{br0k3n_4p4rt...n3ver_t0_b3_r3p41r3d}!

If we run the binary once again and input this string, we get the message Yes! That's right!, which confirms that this is the flag.
