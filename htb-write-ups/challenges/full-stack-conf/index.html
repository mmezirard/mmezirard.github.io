XSS

The website clearly states at the bottom of the page that we can use alert to get the flag, the title of the page is xss, and the challenge description informs us that the administrator logs in regularly to check its emails, so it's crystal clear that we have to perform a stored XSS here.

I'll go with the simplest XSS payload I can think of: <script>alert()</script>.

After a few seconds, the website opens a popup giving us the flag: HTB{p0p..p0p..p0p...alert(1337)}.
